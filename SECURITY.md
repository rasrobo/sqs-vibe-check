# Security

## Reporting a Vulnerability

If you discover a security vulnerability in VibeCheck, please do NOT open a
public GitHub issue. Instead, email security@sqs.chat.

We will acknowledge receipt within 48 hours and provide an estimated timeline
for a fix.

## Responsible Disclosure

We ask that you give us reasonable time to address the vulnerability before
publicly disclosing it.

## Scope

VibeCheck wraps ClamAV (maintained by Cisco Talos). ClamAV-specific
vulnerabilities should be reported through their channels:
https://github.com/Cisco-Talos/clamav
